02 项目介绍  
  
项目描述  
  
一款以Web与全版本服务漏洞检测为核心的辅助性主、被动扫描工具。  
```
集成WAF检测、指纹信息、插件扫描
能够解析Json、XML、伪静态等复杂格式的参数（Beta）
为扫描记录和其他数据提供基于SQLite3的存储支持
基于 Python3 的开源解决方案，支持跨平台
支持IPV6域名解析（Beta版）
```
